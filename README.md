# Fluxer Color Schemes

This repository contains custom color schemes for [Fluxer](https://fluxer.app).

## Available Themes

- [Rosé Pine](themes/rosepine.css)
- [Gruvbox](themes/gruvbox.css)
- [Totally Not Discord](themes/totally-not-discord.css)

## How to apply

1. Open **User Settings**.
2. Go to **Look & Feel**.
3. Expand **Custom Theme Tokens** & scroll down.
4. Replace the existing CSS with the content from one of the `.css` files in this repository.
